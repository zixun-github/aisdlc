## 安全测试用例模板

用于验证安全控制与漏洞防护。

```markdown

**优先级：** P0（关键）
**类型：** 安全
**OWASP 分类：** [A01-A10]
**风险等级：** 关键 | 高 | 中 | 低

### 目标
验证 [安全控制] 能防止 [漏洞/攻击]

### 前置条件
- 具有 [角色] 的测试账号
- 安全测试工具已配置
- 审计日志已启用

### 测试步骤

1. 尝试 [攻击向量]
   **输入：** [恶意载荷]
   **预期：** 请求被拦截/净化

2. 验证安全控制响应
   **预期：**
   - 错误信息：通用（无信息泄露）
   - 日志记录：攻击尝试已记录
   - 账号：未被攻破

### 攻击向量
- [ ] SQL 注入
- [ ] XSS（存储/反射）
- [ ] CSRF
- [ ] 认证绕过
- [ ] 权限提升

### 合规检查
- [ ] [法规] 要求满足
- [ ] 审计轨迹完整
- [ ] 数据已加密
```

## 优先级定义

| 优先级 | 说明 | 执行时机 |
|--------|------|----------|
| P0 | 关键路径，阻塞发布 | 每次构建 |
| P1 | 主要功能，高影响 | 每日/每周 |
| P2 | 标准功能，中等影响 | 每周/发布前 |
| P3 | 次要功能，低影响 | 仅发布前 |
